   🆘 SI TIENES PROBLEMAS
════════════════════════════════════════════════════════════════════

1. "Node no está instalado"
   → Descarga de https://nodejs.org/ (LTS version)

   → npm install -g @angular/cli@latest

3. "Puerto 4200 en uso"
   → ng serve --port 4300

4. "Error en npm install"
   → Elimina node_modules y package-lock.json
   → npm install nuevamente
